# GamesenseReloadedFix

Исправление бага в **Clients (NL,GS, etc.)**, при запуске которых вместо **CS:GO Legacy** запускается **CS2**.

## Возможности

- Поиск Steam через Windows Registry
- Сохранение пути к Steam для будущих запусков
- Запуск игры через Steam

## Для чего

Проект создан как временное решение проблемы, когда loader пытается открыть `cs2.exe`, заместо необходимой вам CS:GO.
